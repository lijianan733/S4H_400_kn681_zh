access condition

Condition in a CDS role defined in an access rule of the statement DEFINE ROLE for a CDS entity. CDS access control evaluates the access conditions of a CDS entity by default when ABAP SQL is used to access objects. Access conditions can comprise PFCG conditions, literal conditions, user conditions, and inheritance conditions.
