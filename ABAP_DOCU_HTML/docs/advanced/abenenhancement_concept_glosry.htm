enhancement concept

Also called Enhancement Framework. Framework for the integration of various concepts dealing with the enhancement of repository objects. The enhancement concept is realized by the Enhancement Builder tool in ABAP Workbench, and in the case of ABAP source code, by using specific ABAP statements.
