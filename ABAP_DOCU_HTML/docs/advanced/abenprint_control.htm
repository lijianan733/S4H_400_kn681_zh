Spool Control

Configuring the Upper and Left Margins

The statement SET MARGIN can be used to define the upper and left margins of a spool list.

Archiving Information in Spool Lists

When spooling is active, the statement PRINT-CONTROL can be used to write formatting information for optical archiving to the spool system.

Obsolete Formatting of Spool Lists

When spooling is active, an obsolete variant of the statement PRINT-CONTROL can be used to write control statements for output devices (printers) to the spool system.
