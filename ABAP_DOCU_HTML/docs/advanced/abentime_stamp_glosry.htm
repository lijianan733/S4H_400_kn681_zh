time stamp

Combined date/ time for identifying a point in time. Time stamps in the ABAP runtime environment are POSIX time stamps independent of time zone. Time stamps are represented by a time stamp field with a time stamp type or as time stamps in packed numbers whose data type is defined using the data elements TIMESTAMP or TIMESTAMPL from ABAP Dictionary.
