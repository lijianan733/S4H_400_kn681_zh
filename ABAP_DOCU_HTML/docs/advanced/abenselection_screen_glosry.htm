selection screen

Special dynpro that can be defined without Screen Painter by using the statements SELECTION-SCREEN, PARAMETERS, or SELECT-OPTIONS in the global declaration part of executable programs, function groups, and module pools. Selection screens are processed by the ABAP runtime environment, which raises selection screen events. Both standard selection screens and standalone selection screens are possible.
