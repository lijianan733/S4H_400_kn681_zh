Branches

This section describes the branches defined using the following:

  • IF - ENDIF
  • CASE - ENDCASE
  • CASE TYPE OF - ENDCASE

Special types of branch are group level processing for internal tables and extracts and branches for exception handling.

Note

The conditional operators COND and SWITCH can be used to implement branches in operand positions.
