The following sections summarize the built-in SQL functions supported by ABAP CDS and ABAP SQL, across all platforms.

  • SQL functions for numeric values
  • SQL functions for strings

Descriptions of the potential operands and data types can be found in the corresponding documentation for ABAP CDS and ABAP SQL.

Note

Some SQL functions are also implemented for use in the table buffer on AS ABAP. Table buffering is not bypassed when these functions are used.
