selection screen version

Version of the standard selection screen of a logical database. Selection screen versions allow certain elements of the standard selection screen to be suppressed. A selection screen version is created using SELECTION-SCREEN - VERSION in combination with SELECTION-SCREEN EXCLUDE and can be selected when associating an executable program with a logical database.
