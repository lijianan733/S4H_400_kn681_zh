ABAP-managed HDI container

AMHC for short. Special HDI container managed by AS ABAP. An AMHC is created and modified using transaction SCTS_AMHC. SAP HANA Transport for ABAP (HTA) provides an AMHC and its HDI objects with an interface to the Change and Transport System. When an AMHC is created, a logical HDI container is defined. A physical HDI container is then created for this container in the SAP HANA database when it is activated. The logical HDI container can be used as a logical schema in an AMDP macro. Here it stands for the database schema of the physical HDI container.
