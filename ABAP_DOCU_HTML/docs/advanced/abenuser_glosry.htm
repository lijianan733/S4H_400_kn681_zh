user

1. Defined in AS ABAP by a user name and a user master record in transaction code SU01 .

2. User (person or other system) that is logged on to the AS ABAP.

Once users have logged on to AS ABAP and have been authenticated, they can access functions and objects in AS ABAP during their user session as defined by their authorizations.
