UTF-16

16-bit Unicode character representation. In this representation, all characters are displayed using 2 bytes. The characters in the surrogate area are addressed by a surrogate. UTF-16 is the system code page of a Unicode system. The ABAP programming language supports UCS-2 as a subset of UTF-16 without surrogates.
