exclusive lock

Lock that does not allow other users to set simultaneous locks for the data in question. The ABAP SQL statements INSERT, DELETE, MODIFY, and UPDATE, SELECT with the addition FOR UPDATE plus all corresponding Native SQL statements or statements in AMDP methods set appropriate database locks on the rows they address. As an SAP lock, an exclusive lock is set by calling a lock function module with the appropriate parameters.
