Input Helps on Dynpros

An ABAP Dictionary search help can be bound directly to a dynpro field. Here, the first parameter of the search help is associated with the dynpro field. Only values from the proposal list of the screen can be transported.

Note

Obsolete input helps on dynpros are possible using the additions VALUES and SELECT of the dynpro logic statement FIELD.
