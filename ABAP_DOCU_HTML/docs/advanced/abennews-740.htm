Changes in Release 7.40 and Its SPs

ABAP Release 7.40 is based on 7.0, EhP3 or 7.3, EhP1. Therefore, apart from a few enhancements, this release is based almost in full on 7.0, EhP2. The changes in Release 7.40 are delivered in support packages that are coupled with kernel releases.

The following table shows the support packages and the associated kernel releases for which there are changes in the ABAP programming language:

Support Package Kernel Release
7.40, SP10 7.42
7.40, SP08 7.42
7.40, SP05 7.41
7.40, SP02 7.40
