DELETE DYNPRO

This statement is for internal use only.
It must not be used in application programs.

Syntax

DELETE DYNPRO f.

Effect

Deletes the dynpro with the name specified in the field f.

System Fields

sy-subrc Meaning
0 The dynpro was deleted.
4 The dynpro does not exist.

The content of f consists of the 40 character program name and the four character dynpro number.

Example

DATA DYNPRONAME(44) VALUE 'SAPTEST'.
DYNPRONAME+40 = '0100'.
DELETE DYNPRO DYNPRONAME.
