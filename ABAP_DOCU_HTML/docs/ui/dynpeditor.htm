Editor commands:
----------------

T(OP) Position on first page
B(OTTOM) Position on last page
A(TTACH) nnn
Position on specified line

N(EXT) <(<string>
Search again for string
The line containing the string becomes
the new top line.
If no parameter is set, the parameter of the
last ATTACH or NEXT command
is used.
RENUM(BER) Renumber

FIND string
The string is searched for after the second
displayed line and positioned on the line
found.
If the string contains blanks, it is set
in special characters (for example, - ).

REPL(ACE) s1 s2
The string 's1' is replaced everywhere by the
string 's2'. The length of the strings
can be different.
If a string contains blanks, both strings are
set in special characters
for example, REPL -a b-cd-
