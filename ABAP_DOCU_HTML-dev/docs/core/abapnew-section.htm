NEW-SECTION

Quick Reference

Obsolete Syntax

NEW-SECTION.

Effect

This statement (which cannot be used in classes) is used to create a new spool request, has the same function as the following statement, and is replaced by this statement:

NEW-PAGE PRINT ON NEW-SECTION.

Note

The obsolete statement does not allow the mandatory spool parameters to be specified.
