FREE - no_id

Obsolete Syntax

FREE MEMORY.

Effect

If the addition ID is not specified in the statement FREE MEMORY, all data clusters are deleted from the ABAP memory.

Note

This short form is extremely error-prone and forbidden in classes, since a FREE statement without ID affects all data clusters.
