data source

  • In a ABAP SQL query or a query in ABAP CDS, a data source is a database table, a CDS entity , a classic view or a different resource such as a common table expression, a hierarchy, a table function, or an internal table.
  • In a streaming context, a data source is a repository from which a reader data stream takes its data. A data source can be an external resource such as an LOB column of a database table or a file, or an internal resource such as a string or an internal table. Streaming is currently implemented for database tables, strings, and internal tables.
