assignment operator

Operator for assigning the content of a data object to a variable. In ABAP, the general assignment operator is the equals sign (=). The casting operator (?=) is a special assignment operator. Alongside this are the addition assignment operator (+=), the subtraction assignment operator (-=), the multiplication assignment operator (*=),the division assignment operator (/=), and the concatenation assignment operator (**=) for calculation assignments.
