Only Unicode Systems in Release 7.50

From Release 7.50, AS ABAP can only run as a Unicode system. Systems with non-Unicode code pages are no longer supported. Therefore, all runnable ABAP programs have the ABAP language version Standard ABAP (Unicode) or higher. Programs with the language version Obsolete ABAP (Non-Unicode) can no longer be executed. The syntax check can now only run using the rules from the Unicode check. The ABAP language version Obsolete ABAP (Non-Unicode) can no longer be used.
