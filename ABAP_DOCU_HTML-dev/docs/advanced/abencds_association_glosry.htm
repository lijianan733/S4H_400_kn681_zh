CDS association

Association in ABAP CDS. CDS associations can be defined for CDS views and abstract CDS entities using the addition ASSOCIATION in the CDS DDL in ABAP CDS. Associations define join relationships between source data sources and target data sources. They can be evaluated in CDS DDL, CDS DCL, and in ABAP SQL using SQL path expressions.
