global class

Class that is visible throughout the entire AS ABAP. Global classes are declared in class pools and are flagged as global classes here using the addition PUBLIC of the statement CLASS. A class pool can contain just one global class. All global classes of an AS ABAP are stored together with the global interfaces and data types in ABAP Dictionary in one namespace.
