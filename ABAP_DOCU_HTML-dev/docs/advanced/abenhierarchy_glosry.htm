hierarchy

Tabular data source in ABAP SQL whose rows represent hierarchy nodes that themselves are joined using hierarchy edges. Special hierarchy columns contain hierarchy attributes of the individual hierarchy nodes in the hierarchy. Hierarchies are created either by a hierarchy generator or an existing CDS hierarchy or CTE hierarchy is used. Hierarchies can be used in the FROM clause and as the source of special hierarchy navigators.
