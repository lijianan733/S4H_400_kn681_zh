Processing Lists in the List Buffer

The following statements edit lists that are already saved as screen lists in the list buffers:

  • READ LINE,
  • MODIFY LINE,
  • SCROLL LIST, and
  • DESCRIBE LIST

Spool lists that are sent page by page to the SAP spool system cannot be edited in this manner.
