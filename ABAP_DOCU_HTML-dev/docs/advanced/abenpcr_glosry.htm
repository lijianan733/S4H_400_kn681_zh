parent-child relationship

PCR for short. A parent-child relationship is a directed hierarchy edge between two hierarchy nodes. It defines the type of the nodes. A hierarchy node with outbound edges is a parent node. A hierarchy node with inbound edges is a child node. A hierarchy node with no inbound edges is a root node or an orphan node. A hierarchy node with no outbound edges is a leaf node.
