SAP HANA Studio

Eclipse-based tool for database objects in the SAP HANA database, more specifically for XSC objects saved in SAP HANA Repository. In XSA, SAP HANA Studio is replaced by SAP Web IDE for SAP HANA and SAP HANA Cockpit.
