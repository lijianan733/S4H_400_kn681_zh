BAdI Reference Variables

Special object reference variable declared using a reference to a BAdI and used to access the associated BAdI object. BAdI reference variables are supplied with a BAdI reference by GET BADI and are only used in CALL BADI to call BAdI methods. Technically, BAdI reference variables are class reference variables of the static type of a concrete BAdI class, or of its abstract superclass CL_BADI_BASE.
