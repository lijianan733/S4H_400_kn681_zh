dynpro sequence

The sequence in which dynpros are processed whose flow is determined by the next dynpro for each dynpro in the sequence. The first dynpro in a dynpro sequence is the initial dynpro. Dynpro sequences are called using transaction codes or CALL SCREEN. A dynpro sequence always part of a single popup level and is executed in a single GUI window. A dynpro sequence is ended when the next dynpro with the dynpro number 0 is called.
