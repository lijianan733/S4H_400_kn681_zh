data element

Repository object in ABAP Dictionary that defines an elementary data type or a reference type. As well as the technical type attributes, it describes the semantic meaning of a (database) field (for example, the header for table displays or documentation text). Properties such as data type, length, and so on, are defined directly in the data element or are passed from a domain.
