BAdI interface

A BAdI interface is a global interface that integrates the tag interface IF_BADI_INTERFACE. A BAdI interface can be used as a part of the definition of one or more BAdIs, and may only contain methods (BAdI methods) and events, but not variable attributes. Furthermore, a BAdI interface cannot contain other component interfaces (except for other BAdI interfaces). An interface that contains a BAdI interface is also a BAdI interface.
