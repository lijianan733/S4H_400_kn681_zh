ABAP session

Instance for a user session in AS Instance with ABAP Memory as a separate memory area. Each GUI window of a user session is associated with its own ABAP session. A maximum of 16 ABAP sessions can be managed for a single user session. An ABAP program is executed in an internal session of an ABAP session.
