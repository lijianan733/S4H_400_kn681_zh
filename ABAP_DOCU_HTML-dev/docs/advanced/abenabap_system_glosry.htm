ABAP system

An ABAP Platform implemented using a client-server system consisting of

  • ABAP-specific components of the presentation layer , such as SAP GUI .
  • Application Server ABAP (AS ABAP)
  • ABAP database schema in the standard AS ABAP database containing system programs and system tables plus application programs and application tables.

The name of an ABAP system matches the system ID of its AS ABAP.
