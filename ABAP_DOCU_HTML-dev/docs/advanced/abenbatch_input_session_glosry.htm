batch input session

Groups a series of transaction calls together with input data and user actions. A batch input session can be used to execute dialog transactions in batch input, where some or all of the screens are processed by the session. Batch input sessions are saved persistently and are managed by the batch input manager in transaction SM35. See also batch input tables.
