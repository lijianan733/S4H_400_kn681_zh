UTF

Abbreviation of Universal Transformation Format. Different representations of the Unicode character set. All UTF character representations are based uniquely on the characters defined in Unicode and can be transformed to each other. The representations differ only in the type of addressing. See UTF-8, UTF-16, and UTF-32 plus UCS-2.
