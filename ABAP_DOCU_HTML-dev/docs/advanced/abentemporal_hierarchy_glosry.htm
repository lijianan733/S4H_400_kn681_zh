temporal hierarchy

Special hierarchy containing only hierarchy nodes that meet certain conditions for time intervals. When the hierarchy is created with the addition PERIOD in ABAP SQL or PERIOD in ABAP CDS, columns of the data source of the hierarchy are defined as interval limits called periods and a time interval is specified for the root node set. Validity intervals are determined as intersections of these time intervals. Only nodes with non-empty validity intervals are added to the hierarchy. Here, the validity intervals of child nodes are produced from their period and the validity interval of the parent node.
