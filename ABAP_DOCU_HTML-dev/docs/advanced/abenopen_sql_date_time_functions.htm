ABAP SQL - Date Functions and Time Functions

ABAP SQL supports the following date functions and time functions:

  • Date functions
  • Time functions
  • Time stamp functions
  • Time zone functions
  • Date/time conversions

Note

The date, time, and time stamp types DATN, TIMN, and UTCLONG are not currently supported.
