ABAP Dictionary

Persistent repository for data types that are visible in all repository objects. Among other things, the database tables and views in the standard database plus lock objects are managed in ABAP Dictionary. The classic objects in ABAP Dictionary are edited using the tool of the same name in ABAP Workbench. Called using transaction SE11. ABAP Dictionary also manages the CDS objects from ABAP CDS that are defined in CDS source code in the ABAP Development Tools (ADT).
