loop

Control structure that performs an iteration. A loop consists of a statement block defined by the keywords (DO - ENDDO, WHILE - ENDWHILE, LOOP - ENDLOOP, PROVIDE - ENDPROVIDE, SELECT - ENDSELECT) and that can be executed more than once.
