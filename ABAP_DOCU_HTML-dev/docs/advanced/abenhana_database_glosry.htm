SAP HANA database

The SAP HANA database is the in-memory database of SAP HANA. It can be used as the database for an AS ABAP. In this case, HANA views can be accessed in ABAP programs with ABAP SQL, provided that these views are defined as external views in ABAP Dictionary. Furthermore, AMDP can be used to manage and call database procedures written in SQL Script.
